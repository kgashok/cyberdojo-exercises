PROBLEM STATEMENT 

Stage 1 
Convert a sorted array to binary search tree.

Stage 2

Convert a sorted array to balanced binary search tree. 

For example, for input of [1, 2, 3, 4, 5, 6, 7], the tree 
returned by function must be equivalent of the following tree:

    __4__
   /     \
  2       6
 / \     / \
1   3   5   7
