Write a program that does binary search on a list of numbers. 
  - The program should return the index at which it
    found the number to be searched. 
  - It should be coded as a function having only one return statement 
  

SIMPLIFICATION: 

Write a program that does binary search on a list of numbers. 
  - If present, return True
  - If absent, return False
