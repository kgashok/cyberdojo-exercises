
## Problem Description

![harshad](https://www.dropbox.com/s/e9xqkdfj4luv69n/Skava11.JPG?raw=true)

Write program to check whether a number is Harshad or strong Harshad.

Example:

Input: `201`
Output: True (for strong Harshad)

Input: `12`
Output: True (for Harshad alone)
